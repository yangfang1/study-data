# 数据库基础

## 知识体系

### 1. 关系型数据库
#### MySQL
- 基础概念
  - 数据类型
  - 表结构设计
  - 主键与外键
- 索引优化
  - B+树索引原理
  - 索引类型
  - 执行计划分析
- 事务处理
  - ACID特性
  - 事务隔离级别
  - 锁机制
- 性能优化
  - SQL优化
  - 配置优化
  - 分库分表

#### PostgreSQL
- 特色功能
  - JSON数据类型
  - 全文搜索
  - 地理信息处理
- 高级特性
  - 继承
  - 分区表
  - 并行查询

### 2. NoSQL数据库
#### MongoDB
- 基础操作
  - CRUD操作
  - 文档模型
  - 查询语法
- 高级特性
  - 聚合管道
  - 索引设计
  - 分片集群
- 性能优化
  - 查询优化
  - 索引优化
  - 硬件优化

#### Redis
- 数据类型
  - String
  - Hash
  - List
  - Set
  - Sorted Set
- 高级特性
  - 事务
  - 发布订阅
  - Lua脚本
- 持久化
  - RDB
  - AOF
- 集群方案
  - 主从复制
  - 哨兵模式
  - 集群模式

### 3. ORM工具
#### Sequelize
- 模型定义
- 关联关系
- 查询接口

#### TypeORM
- 装饰器使用
- 实体关系
- 查询构建器

#### Prisma
- Schema定义
- 查询API
- 关系处理
- 迁移工具

#### Mongoose
- Schema设计

## 学习路径

### 入门阶段
1. 学习SQL基础语法
2. 掌握基本的CRUD操作
3. 了解数据库设计原则
4. 熟悉一种ORM工具

### 进阶阶段
1. 深入理解索引原理
2. 掌握事务处理机制
3. 学习性能优化技术
4. 实践分库分表方案

### 高级阶段
1. 设计高可用架构
2. 实现读写分离
3. 优化大规模数据
4. 处理复杂查询
