# 架构设计原则

## 一、基本原则
### 1. SOLID原则
- 单一职责原则(SRP)
- 开闭原则(OCP)
- 里氏替换原则(LSP)
- 接口隔离原则(ISP)
- 依赖倒置原则(DIP)

### 2. 其他原则
- DRY(Don't Repeat Yourself)
- KISS(Keep It Simple, Stupid)
- YAGNI(You Aren't Gonna Need It)
- LoD(Law of Demeter)
- CRP(Common Reuse Principle)
